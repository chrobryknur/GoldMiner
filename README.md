# GoldMiner
![](https://i.imgur.com/UoBNi1E.png)

### Requirements:  
GTK 3.19+

```
sudo apt-get install libgtk-3-dev  
git clone --depth 1 https://github.com/chrobryknur/GoldMiner.git  
cd Projekt-Marcin-Dabrowski
```  
### Compile the game using: 
```
make
```
### Run the game using:
```
./Game
```
